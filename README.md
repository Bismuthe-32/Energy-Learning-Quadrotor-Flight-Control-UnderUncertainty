# Adaptive Energy Based Learning for Quadrotor Flight Control Under Dynamic Uncertainty

This repo covers the use of an enegy-based technique that uses neural thermodynamic modelling to control quadrotor flight under dynamic undertainty. Current flight control techniques include PID, and LQRs, which only work short-term, and do not stabilize under continuous, changing distrubances. We introduce an energy-based neural technique in stabilizing quadrotor control under dynamic undertainty.
